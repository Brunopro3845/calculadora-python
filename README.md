#  Calculadora em Python

Uma calculadora desenvolvida em Python para realizar operações matemáticas diretamente pelo terminal (CLI).

##  Sobre o Projeto

Este projeto foi criado com o objetivo de praticar conceitos fundamentais de programação em Python, incluindo funções, estruturas condicionais, tratamento de exceções e interação com o usuário por meio do terminal.

##  Funcionalidades

* ✅ Soma
* ✅ Subtração
* ✅ Multiplicação
* ✅ Divisão
* ✅ Potenciação
* ✅ Raiz quadrada
* ✅ Histórico de cálculos
* ✅ Tratamento de divisão por zero
* ✅ Validação de entrada do usuário
* ✅ Interface simples via terminal (CLI)

## 🛠 Tecnologias Utilizadas

* Python 3
* Biblioteca `math`
