# Gerador de Senhas Seguras 

Projeto em Python que gera senhas aleatórias com tamanho definido pelo usuário.

---

##  Funcionalidades
- Geração de senhas aleatórias
- Definição de tamanho pelo usuário
- Uso de letras, números e símbolos

---

##  Tecnologias
- Python
- random
- string

---

##  Como executar
```bash
python main.py
